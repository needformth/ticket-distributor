# 🎫 Распределитель билетов

Простое десктопное приложение для случайного распределения экзаменационных билетов между студентами.  
Позволяет ввести список студентов вручную или загрузить из CSV/Excel, указать количество билетов и получить равномерное распределение (некоторые студенты могут получить на один билет больше).


## ✨ Возможности

- Ручной ввод студентов (каждый с новой строки)
- Загрузка списка из CSV или Excel (с заголовком или без)
- Сохранение результатов в Excel
- Поддержка тёмной темы
- Корректная работа Ctrl+V / Ctrl+X / Ctrl+A независимо от раскладки клавиатуры

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- Зависимости указаны в `requirements.txt`

### Шаги
```bash
# Клонировать репозиторий
git clone https://github.com/needformth/ticket-distributor.git
cd ticket-distributor

# Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate   # Linux/macOS

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python main.py
